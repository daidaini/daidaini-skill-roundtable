# ljg-skill-roundtable

结构化圆桌讨论 — 一个 Claude Code Skill，以"求真"为目标，邀请代表不同思想的真实人物进行多轮辩证对话。

## 功能

- 根据议题自动选择 3-5 位真实历史/当代人物，覆盖多元立场
- 主持人引导多轮辩证讨论，每轮聚焦最深的争议点
- 每轮生成 ASCII 思考框架图（矩阵/光谱/因果环路/层级树）
- 讨论结束后生成完整知识网络和开放问题列表
- 全程输出为 org-mode 文件，保存到 `~/Documents/notes/`

## 安装

```bash
/install lijigang/ljg-skill-roundtable
```

## 使用

```
/ljg-roundtable 人工智能是否拥有真正的创造力？
圆桌讨论 自由意志是否存在？
圆桌 经济增长的极限在哪里？
```

### 讨论中的指令

| 指令 | 含义 |
|------|------|
| `可` | 接受下一层问题，继续推进 |
| `止` | 结束讨论，生成知识网络 |
| `深入此节` | 围绕当前争议点继续深挖 |
| `引入新人物` | 指定新人物加入讨论 |

## 设计原则

- **求真 > 和谐**：鼓励尖锐交锋，拒绝表面共识
- **挖深不铺广**：每轮只追一条最深的裂缝
- **忠于真实思想**：每位人物严格基于其经典著作和思想体系发言
- **结构可视化**：用 ASCII 拓扑图暴露讨论的深层结构
